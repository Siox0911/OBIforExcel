# Office Barcode Integration for Excel

[Deutsche Beschreibung](#German)

## English
Office Barcode Integration for Excel is a tool to integrate barcodes in a Excel workbook.

_Current Version 1.1.0.X_

[changelog.md](changelog.md) for more details

### Features

- Insert a DataMatrix Code into a cell
- Cells are connected to the DataMatrix code, cell changes also lead to a new DataMatrix code
- The entire workbook in Excel is monitored, as well as linked cell changes across worksheets, 
which results in changes to the DataMatrix code, as long as a cell is linked to the DataMatrix 
code in the same worksheet.
- The generated DataMatrix codes are images, so you can easily copy and use them from Excel.
However, the connection to the cell in Excel is lost.
- It is possible to drag any image into the workbook and connect it to a cell. The image 
serves as a kind of placeholder for size, position and format. More in the video, 
which you should look at.

## German
Office Barcode Integration for Excel ist ein Werkzeug zur Erzeugung von Barcodes in einer
Excel Arbeitsmappe.

_Aktuelle Version 1.1.0.X_

[changelog.md](changelog.md) für mehr Details

### Funktionen

- Füge einen DataMatrix zum Tabellenblatt hinzu
- Die Zellen sind verbunden mit dem DataMatrix, Zellenänderungen führen automatisch zur 
Änderung des DataMatrix-Codes
- Die gesamte Arbeitsmappe in Excel wird überwacht, ebenso wie verknüpfte Zellenänderungen 
über Arbeitsblätter hinweg, was zu Änderungen am DataMatrix-Code führt, sofern eine Zelle 
mit dem DataMatrix-Code im selben Tabellenblatt verbunden ist.
- Die generierten DataMatrix-Codes sind Bilder, diese können also auch in anderen Anwendungen 
verwendet werden. Dabei geht allerdings die Verbindung zur Zelle in Excel verloren.
- Es ist möglich, ein beliebiges Bild in die Arbeitsmappe zu ziehen und es mit einer Zelle 
zu verbinden. Das Bild dient als eine Art Platzhalter für Größe, Position und Format. Mehr 
im Video, das du dir anschauen solltest.
